# nds2dsiware
As DsiWARE public.sav are just truncated FAT12 filesystems this tool generates all necessary files to use .nds NO-INTRO roms with HiyaCFW. Linux-only, not macOS.

Usage:
```
./convert_to_dsiware.bash <.nds file name>
```
